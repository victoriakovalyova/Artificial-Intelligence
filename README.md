# World Model + VLM Scorer for MiniGrid

Демо-проект, объединяющий RSSM-модель мира (Dreamer/PlaNet style) и VLM-оценщик (CLIP) для управления агентом в среде MiniGrid.

**Среда:** `MiniGrid-Empty-8x8-v0`  
**Модель мира:** RSSM с детерминированным (GRU) и стохастическим состояниями, обучена на 100 случайных эпизодах (30 эпох).  
**VLM:** `openai/clip-vit-base-patch32`, текстовая цель `"a red triangle agent reaching a green goal"`.  
**Планирование:** MPC со случайной стрельбой (100 кандидатов, горизонт 10).

## Результаты

Оценка проводилась на **200 эпизодах** (seed = 42, максимальная длина эпизода 100 шагов).

| Метод                          | Success Rate |
|--------------------------------|--------------|
| Random                         | 0.165        |
| World-Model Planning (no VLM)  | 0.470        |
| **World-Model Planning + VLM** | **0.680**    |

## Обсуждение

### Основные failure modes
- **Накопление ошибок модели мира** – на горизонте 10 воображаемый агент часто «дрейфует», планировщик выбирает неоптимальные действия.
- **Слабое разрешение VLM** – CLIP с трудом различает маленький красный треугольник на сетке 7×7, давая высокий скор даже при неудачной позиции.
- **Случайный поиск** – 100 сэмплов иногда недостаточно, чтобы найти удачную последовательность (особенно если цель далеко).

### Future work (при большем времени)
- Улучшить модель мира: увеличить ёмкость, использовать трансформерное состояние (DreamerV3).
- Применить CEM (Cross-Entropy Method) для более эффективного планирования.
- Дообучить VLM на доменных кадрах MiniGrid или использовать более мощную модель (BLIP-2, LLaVA).
- Добавить штраф за время в objective планировщика, чтобы избежать зацикливания.
- Интегрировать actor-critic (полный Dreamer) для быстрого исполнения без онлайн-планирования.
